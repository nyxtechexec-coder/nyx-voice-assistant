# Recommended Tech Stack

## Frontend
- Flutter
- Riverpod or Bloc
- secure local storage
- audio recording/playback packages
- websocket client

## Backend
- FastAPI
- PostgreSQL
- Redis
- Celery or background worker framework
- Nginx
- Docker

## AI Services
- Deepgram for speech-to-text
- ElevenLabs for text-to-speech
- OpenAI or equivalent reasoning model for assistant orchestration

## Infra
- VPS or cloud deployment
- Docker Compose initially
- later Kubernetes optional
