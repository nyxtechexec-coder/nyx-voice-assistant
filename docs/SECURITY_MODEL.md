# Security Model

## Principles
- least privilege
- explicit user approval for sensitive actions
- full auditability
- encrypted communication
- revocable access

## Sensitive Actions Requiring Approval
- sending messages
- deleting data
- spending money
- external publishing
- credential usage
- device control actions

## Security Controls
- token-based auth
- encrypted storage for secrets server-side
- signed sessions
- action logs
- permission scopes per integration
- secure device session management
